cross><talk — DJ Lab 🎧
A Pioneer CDJ-inspired two-deck DJ app that runs entirely in your browser — no install, no account, no internet required after loading.
Load two tracks, beatmatch them, and blend with a crossfader. Built as a single HTML file that works as a home screen app on iPhone and iPad via Safari. 🎵

What it does 🎛️
Two full decks, each with:

📊 Waveform display with seekable playhead and cue point marker
🔢 BPM detection — auto-detects tempo on load
⏺️ 4 hot cue pads — tap to save a position, tap again to jump back instantly
🎡 Jog wheel — drag to nudge tempo for fine beatmatching
⏱️ Phase indicator — shows when both decks are beat-aligned
🔗 Sync meter — live BPM match percentage between decks
📈 VU meter with peak clip indicator
🎧 PFL (solo monitor) — isolate one deck to check it before mixing in

Mixer section:

🎚️ Individual channel faders for each deck
↔️ Crossfader with smooth touch and mouse support
🎛️ Filter and Echo effects per deck, with wet/dry toggle


Aesthetics 🖤🟠
Styled after the Pioneer CDJ-2000NXS2 and DJM-900NXS2 — near-black chassis, orange accent controls, green LCD displays with CRT scanline texture, cyan and magenta channel colours. Every LED, every jog wheel rim light, every button response is built to feel like the real hardware.

Install on iPhone or iPad 📱

Open the GitHub Pages link in Safari
Tap Share → Add to Home Screen
Opens full-screen, no address bar — feels like a native app


Tech 🛠️
Pure HTML, CSS, and JavaScript. No frameworks, no dependencies, no build step. Uses the Web Audio API for all playback, effects, and analysis. Works offline once cached by Safari.
One file. No server. Just drag, drop, and DJ. 🎶
